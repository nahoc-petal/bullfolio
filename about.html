About Bull

Bull is an open source cryptocurrency portfolio and tracker. The goal is to clearly show you how much money you're holding and what has changed in the last week.

Inspiration

Bull is loosely inspired off different apps, namely Blockfolio and Exodus, which I both use daily. The reason I built Bull is simple: I love building stuff and was looking for a desktop version of I decided to create my very own cryptocurrency tracker. I strongly believe you should build things for yourself first and then make it available to anyone. If you're looking for new features, feel free to contact me or implement them yourself!

Technologies

This is a simple project built off the following technologies and frameworks:


  • * Bulma for the CSS
  • * CoinMarketCap's API
  • * jQuery
  • * Firebase for the database and login flow
